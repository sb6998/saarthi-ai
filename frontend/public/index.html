Fetch the data from the API

Please enter the API in cURL format:

Format for:

GET Request: curl -X GET -H 'Access-Control-Request-Method: GET' -H 'Content-Type: application/json' "http://127.0.0.1:8000/api/books/"

POST Request: curl -X POST -H 'Access-Control-Request-Method: POST' -H 'Content-Type: application/json' "http://127.0.0.1:8000/api/books" -d "{"name":"abc8","isbn":"123-abc","authors":["a"],"number_of_pages":"1","publisher":"a","country":"b","release_date":"2021-11-01"}"

DELETE Request: curl -i -X DELETE -H 'Access-Control-Request-Method: DELETE' -H 'Content-Type: application/json' "http://127.0.0.1:8000/api/books/6"

PATCH Request: curl -i -X PATCH -H 'Access-Control-Request-Method: PATCH' -H 'Content-Type: application/json' "http://127.0.0.1:8000/api/books/11" -d "{"name":"abDDc","isbn":"123-abc","authors":["a","b","c"],"number_of_pages":"1","publisher":"a","country":"b","release_date":"2021-11-01"}"
